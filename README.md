一個兼具**盲翻運氣**、**長程滑行策略**與**連續消除張力**的雙人麻將對弈網頁遊戲。專為手機直式全螢幕操作打造，融合傳統麻將花色與經典「湊十點」消除規則，帶來流暢的單手單屏操控體驗。

---

## 🎮 遊戲核心特色

* **拉霸式動態開局**：首頁「筒、索、萬」三種花色高畫質牌面高速閃爍，選定模式後自動拉霸式減速，隨機抽選本局的主題花色。
* **完美滿配置佈局**：4x9（共 36 格）精緻直式牌盤，開局全面鋪滿 1-9 點各 4 套麻將，畫面緊湊無缺牌。
* **飛車式長程推牌**：打破傳統鄰格限制。牌面翻開產生空格後，亮牌只要在直線軌道上無阻擋，即可移動「任意格數」進行遠程突襲！
* **黃金外框與消牌特效**：兩張亮牌數字相加為 10 點（例如 1+9、4+6）時，外框黃金光芒快速閃爍 3 次後粉碎消失，提供極強視覺回饋。
* **反制拉扯鎖定**：剛被對手移動過的牌會附加「灰色暗影」進入鎖定狀態，下一手無法立即反向推回，大幅增加博弈深度。
* **防消極殘局倒數**：全盤翻開後觸發「15 步進攻倒數」，若連續 15 步無人消除，將啟動和平結算，徹底解決互相迴避的死循環。
* **永續戰績看板**：內建持久化戰績歷史紀錄，結合瀏覽器 `localStorage`，重新整理或關閉網頁戰績也絕不丟失。

## ⚔️ 雙遊玩模式說明

本遊戲提供兩種不同博弈深度的遊戲模式，玩家可在首頁自由選擇：

| 模式名稱 | 核心規則 | 策略特點 |
| :--- | :--- | :--- |
| **基礎模式** | 每次進行「翻牌」或「移動牌」後，無論有沒有得分，**立即強制換手**。 | 節奏沉穩，考驗每一步的防守與卡位，避免留給對手好牌。 |
| **進階模式 (連擊)** | 翻牌或移動牌湊十成功後，可**觸發連擊（Combo）特權再動一手**！直到沒有湊十才換手。 | 節奏極快、爆發力強！考驗玩家的盤面記憶力，提供一波流逆轉勝的機會。 |

> 🤖 **智慧 AI 升級**：在進階模式下，電腦也擁有相同的連擊智慧。它會自動感知連擊權利，並優先尋找場上能直接湊十的長程移動，沒有亮牌時才會繼續盲翻，絕不放過任何連擊機會！

## 🕹️ 遊戲操作流程

### 1. 模式與花色抽選
開啟遊戲後，點擊 **「進入對局 (基礎模式)」** 或 **「進入對局 (進階連擊)」**，首頁三種花色的外框會逐漸減速，最終定格隨機選定一種花色（萬子 / 索子 / 筒子）作為本局卡牌。

### 2. 雙軌操作機制
雙方輪流執手，每回合可從以下兩種動作中**二選一**：
* **盲翻牌**：點擊盤面上任一暗色蓋牌將其翻開。若翻開後與其上下左右「已翻開」的鄰居相加等於 10，即可將兩張牌收為己有（得 2 分）。
* **策略推動**：點擊已翻開的正面亮牌，盤面會亮起綠色路徑提示。點擊同一直線軌道上的任意空格，牌便會流暢滑行過去。滑行到位後若與新鄰居湊成 10 點亦可得分。

### 3. 勝負判定
* 當盤面完全清空，或場上無任何合法移動且暗牌全翻開、或 15 步進攻倒數歸零時，遊戲結束。
* 獲得牌數較多的一方（分數高者）獲勝，對局結果將自動計入下方的「歷史戰績紀錄」。

## 🛠️ 技術架構與亮點

* **純原生技術（Vanilla JS / CSS3 / HTML5）**：無任何第三方重型框架，極速載入。
* **全動態自適應演算法（Responsive Viewport Engine）**：
  捨棄傳統固定 `vh/vw` 限制，採用 JavaScript 即時雙重上限交叉比對公式（模擬最大安全高寬比），確保在各款手機畫面上，**棋盤皆能水平垂直完美撐滿且絕不溢出或切邊**。
* **高性能 CSS3 動態渲染**：
  消牌與平移滑動全面套用 `cubic-bezier(0.25, 1, 0.5, 1)` 動態插值曲線與 CSS 變數（`--tx`, `--ty`），動畫絲滑不掉幀。
* **非阻塞型異步狀態機（Async/Await）**：
  核心邏輯由異步事件驅動，確保「滑行就位 ➔ 判定鄰居 ➔ 外框閃爍 ➔ 粉碎消除 ➔ 連擊判定 ➔ 回合切換」所有視覺回饋順序完美串聯。
