一個兼具**盲翻運氣**與**長程殘局推動策略**的雙人麻將對弈網頁遊戲。專為手機直式全螢幕操作打造，融合了傳統麻將花色與經典「湊十點」消除規則，提供極致流暢的單手單屏操控體驗。

---

## 🎮 遊戲核心特色

* **拉霸式動態開局**：首頁「筒、索、萬」三種花色高速跑馬燈閃爍，點擊進入後自動拉霸式減速，隨機抽選該局的主題花色。
* **完美滿配置佈局**：4x9（共 36 格）精緻直式牌盤，開局全面覆蓋 1-9 點各 4 套麻將，畫面緊湊無缺牌。
* **車／飛車式長程推牌**：打破傳統鄰格限制。牌面翻開產生空格後，亮牌只要直線軌道上無阻擋，即可移動「任意格數」進行遠程突襲！
* **黃金外框閃爍消除**：兩張亮牌數字相加為 10 點（例如 1+9、4+6）時，外框黃金光芒快速閃爍 3 次後粉碎消失，提供極強視覺回饋。
* **反制拉扯鎖定**：剛被對手移動過的牌會附加「灰色暗影」進入鎖定狀態，下一手無法立即反向推回，大幅增加博弈博弈深度。
* **防消極殘局倒數**：全盤翻開後觸發「15步進攻倒數」，若連續 15 步無人消除，將啟動和平結算，徹底解決互相迴避的死循環。
* **永續戰績看板**：內建持久化戰績歷史紀錄，結合瀏覽器 `localStorage`，重新整理或關閉網頁戰績也絕不丟失。

## 🕹️ 玩法說明

### 1. 遊戲起始 (花色抽選)
開啟遊戲後，點擊 **「進入對局」**，首頁三種花色的外框會逐漸減速，最終定格隨機選定一種花色（萬子 / 索子 / 筒子）作為本局卡牌。

### 2. 雙軌操作機制
雙方輪流執手，每回合可從以下兩種動作中**二選一**：
* **盲翻牌**：點擊盤面上任一暗色蓋牌將其翻開。若翻開後與其上下左右「已翻開」的鄰居相加等於 10，即可將兩張牌收為己有（得 2 分）。
* **策略推動**：點擊已翻開的正面亮牌，盤面會亮起綠色路徑提示。點擊同一直線軌道上的任意空格，牌便會流暢滑行過去。滑行到位後若與新鄰居湊成 10 點亦可得分。

### 3. 勝負判定
* 當盤面完全清空，或場上無任何合法移動且暗牌全翻開、或 15 步進攻倒數歸零時，遊戲結束。
* 獲得牌數較多的一方（分數高者）獲勝。

## 🛠️ 技術架構與亮點

* **純原生技術（Vanilla JS / CSS3 / HTML5）**：無任何第三方重型框架，極速載入。
* **全動態自適應演算法（Responsive Viewport Engine）**：
  捨棄傳統固定 `vh/vw` 限制，採用 JavaScript 即時雙重上限交叉比對公式（模擬最大安全高寬比），確保在從舊款 iPhone 到最新折疊屏手機上，**棋盤皆能水平垂直完美撐滿且絕不溢出或切邊**。
* **高性能 CSS3 動態渲染**：
  消牌與平移滑動全面套用 `cubic-bezier(0.25, 1, 0.5, 1)` 動態插值曲線與 CSS 變數（`--tx`, `--ty`），動畫絲滑不掉幀。
* **非阻塞型異步狀態機（Async/Await）**：
  核心邏輯由異步事件驅動，確保「滑行就位 ➔ 判定鄰居 ➔ 外框閃爍 ➔ 粉碎消除 ➔ 回合切換」所有視覺回饋順序完美串聯。
